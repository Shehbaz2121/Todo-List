# Todo-List
A simple To-Do List web application built using HTML, CSS, and JavaScript. Users can add tasks to a list and clear all tasks when needed. The app also includes a limit on the number of tasks that can be added and features a responsive layout for smaller screens.<br>
## Link: https://projectthird.netlify.app
# Features:

-Add tasks to a list

-Clear all tasks with one button

-Task limit to prevent overflow

-Responsive design for mobile screens

-Simple UI using pure HTML, CSS, and JavaScript
